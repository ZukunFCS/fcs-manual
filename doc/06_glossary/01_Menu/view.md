```{rubric} **レイアウトの保存方法**
```

1.【Save current】を押すと「Layout Registration」ウィンドウが出るので、  
 Layout Nameにレイアウトの名前を入力し【Save】を押します。
```{figure} /images/06_glossary_Menu_View_Savecurrent.jpg
:width: 80%
:align: center
```

<br><br>

2.保存したレイアウトが一覧に出ます。  
【Apply】：保存したレイアウトに変更  
【Delete】：保存したレイアウトを削除
```{figure} /images/06_glossary_Menu_View_Layout_02.jpg
:width: 80%
:align: center
```

<br>

3.保存したレイアウトのデータは以下のパスに格納されます。  
```
C:\Users\[ユーザー名]\.fcs\Cortado\[FCSバージョン]\layouts\saved
```
