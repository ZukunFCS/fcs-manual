## 作業フロー

FCSを使用したアニメーション制作の全体的な流れを説明します。

---

1. [**FCS起動**](01_launch-FCS) \
   アプリケーションの立ち上げ方法について説明します。 \
   
2. [**Session作成**](02_create-or-open-session) \
   セッションの新規作成と保存の手順です。 \
   
3. [**Mayaの起動**](03_launch-maya) \
   MayaとFCSプラグインを連携させる設定を行います。 \
   
4. [**Controller登録**](04_register-controller) \
   HMC動画と各種コントローラーの紐付け方法です。 \
   
5. [**Profile作成**](05_create-profile) \
   キャラクター設定（プロファイル）の構築手順です。 \
   
6. [**出力**](06_export-animation) \
   最終的なアニメーションデータの書き出しを行います。 \
   
7. [**Sessionのエクスポート**](08_export) \
   作成したSessionデータを外部へ出力する方法です。

---
